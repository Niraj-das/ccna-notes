# Network

## What is Network?
A network is when two or more device connected with each other and share resources is called Network.

<img width="902" height="281" alt="network" src="https://github.com/Niraj-das/ccna-notes/blob/main/fundamentals/assest/network.png" />

As large as the internet with million of devices

## Function of Networking

- Sharing information (ex: E-mail etc..)
- Sharing file (ex: doc, pdf etc..)
- Sharing resources (ex: printer- Multiple computer can use same printer)
- Sharing applicaiton


## Network component

- Host device
- Server
- Network interface card (Mac)
- Communication link (cable)
- Routers
- Switches
- Firewall
- Network protocol

## Characterstic

- Fault tolerance
- Scalability
- Quality of service
- Security
