# Network Monitoring System

It monitor:
1. Traffic
2. Bandwidth Utilization
3. Uptime
4. Connection

## Benifit
1. Clear visibility into the network
2. Better use of IT resources
3. Early descision into future infrastructure need
4. Ability to identify security threat

## Network Monitoring Protocol

1. SNMP
   - Basically use in many type of device varies from switch to router, printer etc
   - Status of devices and what configuration device are running with call and response
2. Tracert
   - It use to trace the route of traffic moving from one hop to another
