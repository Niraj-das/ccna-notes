# Switch

A switch is the l2 or l3 netwok device. It is a intelligent device that know which packet need to  transfer to which endpoint that connect to switch.
It maintain the MAC table and store the mac address of each endpoint.

<img width="617" height="347" alt="image" src="https://github.com/Niraj-das/ccna-notes/blob/main/fundamentals/assest/Switch.png" />

Mac address of all connected nodes are stored in switch

## Type of switch

### 1. Managed switch
- Configuration can be done
- Perform better
- Security feature
- Expensive

### 2. Unmanaged switch
- No configuration
- Avg. performance
- No security feature
- Cheap

## Broadcast Address
   255.255.255.255
   FF:FF:FF:FF

## Switch maintain Mac table

| Ports    |  Mac address      |
|----------|-------------------|
|   1234   | 2A:4f:1d:4c:5d:5f |
|   2345   | 8D:4B:56:91:2E:AC |
|   4567   | F0:23:9A:7D:5C:11 |
|   5679   | 6C:DA:10:EF:44:9B |

## Note:
It trust only source Mac address
